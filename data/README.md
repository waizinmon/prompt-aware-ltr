# data

Datasets used by the training/eval pipeline. Too large for git — hosted on Google Drive.

Download from: https://drive.google.com/... *(replace with your share link)*

Place these files here:

- `train_predictor.jsonl` — training set for `scripts/0b_train_predictor.py` (~10–24k samples)
- `dataset1_eval.jsonl` — in-distribution eval set, held out
- `dataset2_eval.jsonl` — out-of-distribution eval set, held out

Generated by `scripts/0a_build_dataset.py` from the public LMSYS-Chat-1M dataset
(requires a HuggingFace account and accepting the dataset's usage terms).
